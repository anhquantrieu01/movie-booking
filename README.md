# 🎬 Movie Booking System

A full-stack movie ticket booking web application built with Next.js (App Router).

---

## 🛠 Tech Stack

- Frontend: Next.js, React, Tailwind CSS  
- Backend: Next.js Route Handlers, server action  
- Database: PostgreSQL + Prisma ORM  
- Payment: VNPAY (Sandbox)  
- Deployment: Vercel  

---

## ⚡ Architecture

- Server-side rendering (SSR) & ISR for performance  
- Dynamic rendering for real-time data  
- Prisma ORM for type-safe database access  
- API-based separation between client and server
